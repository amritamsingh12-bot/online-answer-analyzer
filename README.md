📚 AI-Based Descriptive Answer Evaluation System
🚀 Project Overview
The AI-Based Descriptive Answer Evaluation System is a web application designed to automatically evaluate long-form descriptive answers using Artificial Intelligence.
The system analyzes answers based on:
Relevance to question
Content quality
Structure & coherence
Language clarity
Keyword matching
It helps reduce manual evaluation time and provides faster, consistent feedback to students.
🎯 Problem Statement
Manual evaluation of descriptive answers:
Takes significant time
Can be inconsistent
Is difficult to scale for large exams
This project solves these challenges using AI-driven automated scoring.
🛠️ Features
📝 Students can submit descriptive answers
🤖 AI evaluates answers instantly
📊 Score generation based on multiple parameters
👩‍🏫 Teacher review & override option
📂 Question & exam management
📈 Student performance tracking
🔐 Secure login system (Admin / Teacher / Student)
